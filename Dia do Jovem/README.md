## 📌 Dia do Jovem

Projeto desenvolvido com o objetivo de celebrar o **Dia do Jovem (13 de março)**, destacando a importância da juventude na sociedade e incentivando reflexão e criatividade por meio de uma página web.

### 💡 Sobre o projeto

Este projeto foi criado como uma forma de representar o significado do Dia do Jovem, utilizando uma interface simples e visualmente agradável para transmitir uma mensagem positiva sobre o papel dos jovens no presente e no futuro.

### 🚀 Objetivos

* Valorizar a importância da juventude
* Incentivar o protagonismo jovem
* Praticar desenvolvimento web com HTML e CSS

### 🛠️ Tecnologias utilizadas

* HTML
* CSS

### 📷 Funcionalidades

* Página informativa sobre o Dia do Jovem
* Estrutura organizada e responsiva *(se for responsivo, senão remove)*
* Design visual temático

### 🎯 Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados:

* Estruturação de páginas com HTML
* Estilização com CSS
* Organização de layout e design
